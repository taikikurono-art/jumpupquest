01 / ABOUT

JUMPUPクエストとは？

体操教室の練習をRPGゲーム風に記録できるシステムです。お子さんが習得した技をポイントで管理し、成長の軌跡を「冒険者カード」として確認できます。

お子さんができること
  • 🎮 自分のページでスキルを確認
  • 🏆 マスターした技を一覧表示
  • 💬 先生からのメッセージを受け取る
  • 🗺️ ジョブマップで成長をイメージ
保護者ができること
  • 📊 成長レポートを確認
  • 📋 LINE用テキストにコピー
  • 🌟 バッジ・称号を確認
  • 💬 先生コメントを確認
02 / HOW TO START

はじめ方・ログインの仕方

1
クエストのページを開く
先生から共有されたURLをブラウザで開きます。ホーム画面に追加するとアプリのように使えます。
2
「つづきから」をタップ
お子さんの名前が登録済みの場合は「つづきから」→ 名前を検索 → タップでログインできます。
3
マイページを確認する
ログイン後に「マイページ」が表示されます。技の記録・バッジ・先生からのメッセージをここで確認できます。
4
保護者ページへ
マイページ下部の「👪 保護者ページ」ボタンから、成長レポートやLINEコピー機能が使えます。
💡 ヒント： 毎回ログインが必要ですが、お子さんの名前を2〜3文字入れるだけで検索できます。
03 / POINT SYSTEM

ポイントとマスターの仕組み

毎月の練習テストで先生が評価を入力します。3回の試技（チャレンジ）をそれぞれ評価し、ポイントが加算されます。

評価の種類

⭐⭐⭐
できた！
技がしっかりできたとき
+3pt
⭐⭐
あと少し
もう少しでできそうなとき
+1pt
🌱
土台づくり中
まだ練習が必要なとき
+0pt

マスター達成の条件

🏆 マスター条件

3回全部⭐⭐⭐ → 即マスター！（一発マスター⚡）
累計10pt以上 かつ 最後の試技が⭐⭐⭐ → マスター達成

🌱 土台づくりモードについて： 2ヶ月連続で全回🌱になった場合、その技は「土台づくり中」モードになります。ポイントはリセットされず、基礎練習に切り替えるサインです。

ポイントの見方

マイページの「スキル記録」に技ごとのポイントバーが表示されます。10ptでゲージが満タンになりマスター達成です。

04 / JOB SYSTEM

ジョブ（クラス）について

お子さんは最初「ルーキー」からスタートし、技をマスターするにつれてジョブが進化します。ジョブはキャラクターの見た目や得意なジャンルが変わります。

バッジ・称号について

続けた期間や習得した技の数に応じてバッジが自動でもらえます。先生から「あいさつ名人」「あきらめない心」などの姿勢バッジがもらえることもあります。

05 / PARENT PAGE

保護者ページの使い方

マイページ下部の「👪 保護者ページ」から専用ページに移動できます。

📊 成長レポートのダウンロード

「成長レポートをDL」ボタンを押すと、HTMLファイルとしてダウンロードできます。マスター技・挑戦中の技・先生コメント・次のおすすめがまとまっています。

📋 LINE用レポートのコピー

「LINEレポートをコピー」ボタンで、テキスト形式のレポートをクリップボードにコピーできます。そのままLINEに貼り付けて共有できます。

🏠 おうちサポートのヒント

挑戦中の技や土台づくり中の技に応じて、おうちでできる声かけのヒントが表示されます。

💡 おすすめ： レッスン後にお子さんと一緒にマイページを開いて、今日の練習をふり返ってみてください。「これマスターしたの？すごい！」の一言が大きなモチベーションになります。
06 / MESSAGE

先生からのメッセージ

先生からメッセージが届くと、マイページのタイムラインに表示されます。未読のメッセージには🔴マークが付きます。

1
マイページを開く
ログイン後のマイページに「タイムライン」があります。
2
メッセージをタップ
「💬 先生からのメッセージ」の項目をタップすると全文が表示されます。
3
既読になります
タップした時点で既読になります。保護者ページでも最新3件が確認できます。
📅 タイムラインについて： メッセージだけでなく、技をマスターした記録も月ごとにタイムラインに表示されます。成長の記録としてふり返りに使えます。
